## 1. 서비스 개요

### 1.1 제품 정의

캠프닉(CampNic)은 캠핑과 피크닉이 결합된 오프라인 행사에서, 참가자들이 현장 미션을 수행하며 즐길 수 있도록 돕는 웹 기반 플랫폼이다.

### 1.2 핵심 가치

> **"행사 관리자가 노코드로 게임을 만들고, 참가자는 클릭 몇 번으로 즐긴다"**

- **관리자**: 코드나 복잡한 설정 없이 템플릿 기반으로 미션과 보상을 즉시 생성
- **참가자**: 별도 앱 설치 없이 QR/링크로 접속해서 모바일로 미션 수행
- **시스템**: 하나의 시스템에서 여러 행사를 동시에 독립적으로 운영

### 1.3 차별점

- 모바일 우선(Mobile First) 설계, PC도 완벽 지원
- 비밀번호 없는 인증 (전화번호 + SMS 자동 인식)
- 12시간 빠른 입장 토큰
- 행사별 독립 운영 (동시 다중 행사 가능)

## 2. 핵심 컨셉과 사용자

### 2.1 사용자 역할

| 역할 | 설명 | 권한 |
|---|---|---|
| ADMIN | 행사 주관리자 (계정 소유자) | 모든 행사 생성/관리, 모든 데이터 접근 |
| STAFF | 특정 행사에 배정된 운영진 | 배정된 행사만 관리 (승인/모니터링) |
| USER | 행사 참가자 | 자신이 참가한 행사만 접근 |

### 2.2 참가 단위

- **개인 + 팀** 모두 지원
- 행사별로 참가 단위 설정 가능: 개인 전용 / 팀 전용 / 둘 다 허용

### 2.3 동시 다중 행사 운영

한 관리자가 여러 행사를 동시에 만들고 관리할 수 있다. 각 행사는 완전히 독립된 데이터를 가지며, 참가자는 자기가 참가한 행사의 데이터만 볼 수 있다.

| Realtime | Supabase Realtime |
| 배포 | Vercel |
